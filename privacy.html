1. Información que recopilamos

No recopilamos datos personales identificables. La aplicación:

  • No requiere registro ni inicio de sesión.
  • No utiliza cookies de seguimiento ni almacenamiento persistente en servidores.
  • No envía datos a ningún servidor externo más que las llamadas necesarias a nuestra API para el balanceo de cargas.

2. Almacenamiento local

La aplicación utiliza el almacenamiento local (localStorage) de tu navegador para guardar el estado de tus circuitos, asignaciones y configuraciones. Estos datos nunca son transmitidos a nuestros servidores ni a terceros. Puedes borrar estos datos en cualquier momento desde la configuración de tu navegador.

3. Uso de la API

Cuando presionas el botón "Balancear cargas", la aplicación envía a nuestra API los siguientes datos de los circuitos que has ingresado:

  • Tipo de circuito (monofásico, bifásico o trifásico).
  • Corriente (A) o potencia (W) de cada circuito.
  • Fase global seleccionada.

Estos datos se utilizan exclusivamente para calcular el balanceo óptimo y devolver las asignaciones de fase. No se almacenan en nuestros servidores ni se utilizan para ningún otro fin.

4. Servicios de terceros

La aplicación utiliza los siguientes recursos de terceros:

  • Font Awesome (cdnjs) — para los íconos de la interfaz. No recopila datos personales.
  • API de balanceo — alojada en un servicio de hosting. Los datos enviados son efímeros y solo se procesan durante la solicitud.

5. Seguridad

Implementamos medidas de seguridad razonables para proteger los datos durante la transmisión (HTTPS). Sin embargo, ninguna transmisión por Internet es completamente segura, por lo que no podemos garantizar una seguridad absoluta.

6. Tus derechos

Dado que no almacenamos datos personales, no es necesario gestionar solicitudes de acceso, rectificación o eliminación. Sin embargo, si tienes alguna pregunta, inquietud o comentario sobre esta política o sobre el manejo de tus datos, puedes contactarnos a través de los siguientes medios:

  • Correo electrónico: ed648a@gmail.com
  • Repositorio del proyecto: GitHub Issues (enlace pendiente de configuración)

Nos comprometemos a responder a tus consultas.

7. Cambios en esta política

Podemos actualizar esta política de privacidad ocasionalmente. La fecha de la última actualización se indica al inicio de este documento.

📌 Nota: Al ser un proyecto en fase beta, es posible que la política de privacidad se actualice con mayor frecuencia durante el desarrollo inicial.
